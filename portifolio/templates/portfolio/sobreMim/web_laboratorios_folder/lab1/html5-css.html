HTML5 e CSS

HTML5

O HTML5 é a mais recente versão da linguagem de marcação HTML, utilizada para desenvolver páginas web. Com o HTML5, é possível criar páginas mais acessíveis, estruturadas e com novas funcionalidades.

Elementos semânticos Utilizado Descrição
<header> ✔️ Utilizado para o cabeçalho da página (usei logo a seguir ao body).
<nav> ✔️ Utilizado para a navegação da página (Introdução, Localização, Multimedia, Informação).
<main> ✔️ Utilizado para o conteúdo principal da página(usei para colocar a informação relevante na página).
<section> ✔️ Utilizado para dividir a página em seções.
<details> ✔️ Utilizado para ver ou esconder detalhes adicionais (usei para descrição das imagens e tabelas).
<footer> ✔️ Utilizado para o rodapé da página (usei para colocar os meus dados).
<article> ❌ Não utilizei.
<aside> ❌ Não utilizei.
<meter> ❌ Não utilizei.
<progress> ❌ Não utilizei.
<abbr> ❌ Não utilizei.
<address> ❌ Não utilizei.

CSS

O CSS é uma linguagem de estilo utilizada para definir a aparência e o layout das páginas web. Com o CSS, é possível criar designs responsivos, com animações e transições.

Tipo de Seletor Usado Comentário
Seletor de Tipo ✔️ Usado para definir estilos para um tipo específico de elemento HTML, como h1, p, etc.
Seletor de Classe ✔️ Usado para definir estilos para um grupo de elementos com a mesma classe, identificados pelo atributo "class".
Seletor de ID ✔️ Usado para definir estilos para um elemento HTML com um ID específico, identificado pelo atributo "id".
Seletor de Atributo ✔️ Utilizado para selecionar elementos que possuem um atributo específico com um valor exato correspondente.
Seletor Universal ✔️ Usado para definir estilos para todos os elementos, representado pelo asterisco "*".
Seletor de Descendência ✔️ Usado para definir estilos para elementos filhos de um elemento pai específico, por exemplo: div p {}
Seletor de Irmão Adjacente ✔️ Usado para definir estilos para o primeiro irmão adjacente de um elemento específico, por exemplo: h1 + p {}
Seletor de Irmão Geral ✔️ Usado para definir estilos para todos os irmãos de um elemento específico, por exemplo: h1 ~ p {}
Combinador de Seletor Usado Comentário
Descendente ✔️ Usado para definir estilos para um elemento que é um descendente de outro elemento, por exemplo: div p {}
Irmão Adjacente ✔️ Usado para definir estilos para o primeiro irmão adjacente de um elemento específico, por exemplo: h1 + p {}
Irmão Geral ❌ Não utilizado.
Filho Direto ✔️ Utilizado apenas para os elementos que são imediatamente filhos de um determinado elemento, excluindo os elementos netos e outros descendentes mais distantes.
Atributo ✔️ Utilizado para selecionar elementos com base em atributos específicos, como o valor do atributo class ou id
